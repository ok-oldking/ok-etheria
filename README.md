**ok-er** 是一个基于图像识别的 "伊瑟" (Etheria) 自动化辅助工具。本项目旨在通过自动化脚本完成游戏中繁琐的日常任务、副本挂机等操作，解放双手。

本项目仅供学习交流使用，**完全免费**。

## ✨ 功能特性 (Features)

目前支持以下功能（持续更新中）：

* **⚡ 一键收菜**
    * 一键清协会战, 幻音剧场, 暗笼激斗等活动。
    * 一键刷完体力
* **🔄 自动点完成和再来一次**
* **🔄 游戏可后台运行**

## 🛠️ 环境准备

* 支持所有16:9的分辨率：**1920x1080** (缩放 100%)。
* 仅支持PC版伊瑟
* 操作系统：Windows 10 / 11。
